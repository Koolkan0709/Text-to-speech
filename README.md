# Text-to-Speech App

A simple React Native mobile application that takes text input and reads it aloud using the `expo-speech` API.

## Features
* Text input field for scripts or sentences.
* Play and Stop audio controls.
* Built with Functional Components and React Hooks.

## Stack
* React Native
* Expo (`expo-speech`)

## Setup
1. `npm install`
2. `npx expo start`
3. Scan with Expo Go to run on device.
